# Multi-Task Mammography Classification

Проект по классификации маммографических изображений с использованием multi-task learning подхода. Модель одновременно решает несколько задач: определение патологии, классификация типа образования и оценка плотности ткани.

## Структура проекта
```
├── mammography_images/     # датасет с изображениями
│   ├── train/
│   └── test/
├── results/                # результаты экспериментов
├── runs_multibackbone_cv/  # логи обучения
├── experiments.ipynb       # основной ноутбук с экспериментами
└── main.ipynb             # запуск обучения
```

## Основные файлы

- `train_clean.csv` - обработанный датасет для обучения
- `test_clean.csv` - тестовая выборка
- `experiments_multibackbone_cv.py` - скрипт кросс-валидации
- `plot_roc_pr_bootstrap.py` - визуализация метрик

## Модель

Используется EfficientNet B7 в качестве backbone с attention механизмом для фокусировки на важных областях изображения. Multi-task головы позволяют модели учиться более robust представлениям.

## Результаты

Все метрики и графики сохраняются в папке `results/`. Для каждого эксперимента генерируются:
- ROC и PR кривые
- Confusion matrix
- Таблицы с метриками по фолдам

## Зависимости

Основные библиотеки: PyTorch, torchvision, pandas, scikit-learn, matplotlib

## Запуск

Обучение запускается через `main.ipynb` или напрямую через скрипты в командной строке.
